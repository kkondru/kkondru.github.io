Research Experience

  • 2017 - 2024
    Scalable Raft Implementation with Proxy Leaders
    Advisor: Dr R Saranya
    • Developed RaftOptima, an enhanced Raft algorithm addressing scalability issues in large clusters by introducing proxy leaders to delegate broadcasting and acknowledgment tasks, reducing latency by 60% in clusters of up to 25 nodes.
    • Constructed a key-value data store and network simulator to validate the system, demonstrating significant throughput improvements while preserving core safety properties.
  • 2017 - 2024
    Development of Enhanced Raft Leadership Algorithm
    Advisor: Dr R Saranya
    • Developed a Java-based enhanced Raft consensus algorithm resolving symmetric and transitive reachability problems that triggered frequent leader elections.
    • Built a key-value store for replication testing and a custom Discrete Event Simulator to benchmark performance under network partitions, and formally verified leader election safety using TLA+.
